# Study Resources

## Learning the Concepts (read/watch)

| Resource | Best for |
|----------|----------|
| [Python Official Tutorial](https://docs.python.org/3/tutorial/) | Functions, classes, iterators, generators — surprisingly readable |
| [Real Python](https://realpython.com) | Article-per-topic format; great for decorators, comprehensions, `collections`, `pathlib` |
| [CS50P — Harvard](https://cs50.harvard.edu/python/) | Free video course (~10 lectures); good if you prefer watching before coding |

## Practice / Assignments

| Resource | Best for |
|----------|----------|
| [Exercism — Python Track](https://exercism.org/tracks/python) | Structured exercises with automated feedback; best for Week 1 topics |
| [LeetCode](https://leetcode.com) | Filter by Python + Easy; good from Day 7 onward once syntax feels comfortable |
| [Codewars](https://codewars.com) | Kata-style problems; easier to find topic-specific drills than LeetCode |

## Quick Syntax Lookup (when you blank)

| Resource | Best for |
|----------|----------|
| [Python Cheatsheet](https://www.pythoncheatsheet.org) | One page per topic — fast refresher without going deep |
| [W3Schools Python](https://www.w3schools.com/python/) | Quick method signature lookups |

---

## How to use these

1. Read the Real Python article (or official docs) for the day's concept.
2. Close all tabs and solve the day's assignments from memory.
3. Only open the cheatsheet if you're truly stuck — not while solving.
